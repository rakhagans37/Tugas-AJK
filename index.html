Perangkat Perantara, atau juga dikenal sebagai perangkat penghubung (intermediary device), merujuk pada perangkat keras atau lunak yang berfungsi untuk menghubungkan dan mengelola komunikasi antara dua atau lebih perangkat dalam jaringan komputer. Perangkat perantara membantu dalam pengiriman data, routing paket, dan menghubungkan berbagai jaringan atau sub-jaringan. Beberapa contoh perangkat perantara yang umum digunakan adalah:

1. NIC (Network Interface Card)

NIC (Network Interface Card) adalah perangkat perantara yang bertindak sebagai penghubung dengan konversi sinyal yang kuat dan melaluinya dapat terhubung dan berkomunikasi dengan perangkat lain yang juga dapat digunakan oleh komputer itu sendiri. Tujuan utama NIC adalah untuk menghubungkan komputer ke jaringan komputer seperti jaringan area lokal (LAN) atau jaringan yang lebih besar seperti Internet. Kartu jaringan bertindak sebagai antarmuka antara komputer dan jaringan, memungkinkan komputer mengirim dan menerima data menggunakan kabel jaringan atau teknologi nirkabel.



Jenis - Jenis Network Interface Card, NIC ini pada dasarnya terbagi menjadi dua jenis, berikut penjelasan kedua jenis NIC tersebut:


1. Network Interface Fisik / Physical

Network Inteface card fisik ini merupakan sebuah Network interface yang memiliki bentuk fisik, bentuk kartu serta ditancapkan kepada slot didalam motherboard. NIC fisik biasanya digunakan setiap komputer menghubungkan ke dalam jaringan dengan menggunakan kabel pada port RJ-45.

Gambar 2.Network Interface Card (NIC) Fisik

2. Network Interface Logis / Logical

NIC logis ini lawan dari Network interface fisik yang mana merupakan sebuah perangkat lunak atau program yang dibuat untuk dapat mendefinisikan dirinya seolah-olah sebagai Network Interface Card.

Gambar 3.Network Interface Card (NIC) Logis

Fungsi fungsi dari NIC antara lain adalah :

  • Peranti yang menyambungkan kabel jaringan dengan komputer
  • Peranti yang menyediakan pengalamatan secara fisik. Artinya kartu jaringan memiliki kode tertentu yang unik.

2. Router

Router adalah perangkat yang memungkinkan beberapa perangkat bisa terhubung ke internet. Selain itu, Fungsi router yang lainnya adalah mengelola lalu lintas antar jaringan dengan meneruskan paket data ke alamat IP yang dituju. fungsi router Saat ini, router merupakan perangkat yang penting di kantor, kafe, atau rumah. Dengan router, Anda bisa membuat jaringan WiFi sendiri. Perangkat yang membutuhkan koneksi internet, seperti: laptop, smartphone, dan tablet bisa menjalankan fungsinya secara optimal.



Jenis - Jenis Network Interface Card, NIC ini pada dasarnya terbagi menjadi dua jenis, berikut penjelasan kedua jenis NIC tersebut:


1. Router Nirkabel

Router ini menggunakan kabel ethernet untuk terhubung ke modem. Perangkat ini bisa mendistribusikan data dengan mengubah paket data dari kode biner menjadi sinyal radio. Kemudian sinyal disiarkan ke sekitarnya menggunakan antena. Router nirkabel dapat membuat jaringan WLAN (Wireless Local Area Network). Sinyal WLAN ini kemudian dapat digunakan banyak perangkat untuk saling berkomunikasi secara nirkabel.

Gambar 4.Router Nirkabel

2. Router Kabel

Router berkabel menggunakan kabel ethernet untuk terhubung ke modem, sama seperti router nirkabel. Bedanya router ini menggunakan kabel untuk terhubung ke satu atau lebih perangkat dalam jaringan. Router ini bisa membuat jaringan LAN dan menghubungkan semua perangkat ke internet.

Gambar 5.Router Kabel

Fungsi fungsi dari NIC antara lain adalah :

  • Menghubungkan perangkat ke internet
  • Membagikan sinyal
  • Menghubungkan beberapa perangkat
